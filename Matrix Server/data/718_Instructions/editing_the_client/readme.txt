How to make an .exe:
1. Download Launch4j, look it up with a search engine
2. Use the launch4jconfig.xml. Toy with it until it works, you can even
pack all of a java JRE into it and make it completely independent
in Windows only.

How to edit the lobby login interface:
1. Go to Class338.java or look up "changeLogin" as a method. Go to that java file.
2. Toy with the change login function by changing around the size of components.
This is hard coding the client and requires a lot of tinkering. You can change any
interface by hard coding but its usually not recommended.
